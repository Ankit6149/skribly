# Competitor landscape

## Direct/adjacent products identified

- Notezilla — Windows sticky notes with attachment to websites, documents, folders, and programs
- Zhorn Stickies — lightweight Windows sticky notes with attachment features
- TackNote — modern contextual notes on macOS
- Tack — recent small/open-source macOS contextual note concept
- Simple Sticky Notes — mature, free, lightweight traditional Windows sticky notes
- browser extensions that pin notes to webpages and elements

## Validated lesson

The user problem is real and paid products exist. The category is not new.

## Required differentiation

Skribly cannot win as “prettier sticky notes.” It must combine:

- contextual return
- visual annotations beyond rectangular notes
- collapsed low-obstruction markers
- handwriting and drawing
- playful tactile interactions
- modern cross-platform UX
- local-first privacy
- measurable lightness

## Positioning

**A contextual visual annotation layer for your computer**, not another notes notebook.

## Research policy

Before launch, re-check pricing, current feature sets, store reviews, permission models, and release activity. Competitor facts age quickly.
